## 简介
This is a general human language.

Colating language （可乐听语）是一种新的拉丁语族通用语言文字，它以英语为主要设计参考蓝本，俄语、中文、日语、韩语等作为微调，并加入了许多 Colating自创的语文理论元素，从而，克服了在这些语言中所存在的不足之处。

总之，Colating语的诞生，它是建立在自成体系的通用语言文字哲学思想基础之上的产物。其主要内容包含：通用语文创造哲学思想及应用实践，概念级错误释义原理与分析方法，轻松创造新概念标识符等等理论。

• 概念的意义    
概念是构建人类思维（可能是全部地球语音动物，甚至更多）的基础单位，人类思想是建立在概念基础之上的产物。    
人类群体能够掌握有效的概念总数量，是衡量该群体在不同时代进步程度的主要指标之一。    
对于个人而言，只有掌握了足够多有效的概念，以及概念进行链接的逻辑关系，才有可能做到有效地进行思想。

• 语文系统设计的首要任务    
确定宙沏概念对象的总数量：无穷多个；    
确定概念的历史寿命：非本能感知概念至少需要保留如2000年以上，本能感知概念需要保留的年限更长、甚至可以追溯到人类诞生时的起始时代；    
确定人类天生本能语文器官的基本能力。    
宙沏：宇宙内、表、外的一切。

• 概念标识符的本质    
概念图像标识符：概念对象的图像标识编码；    
概念声音标识符：概念对象的声音标识编码。

• 拉丁语单词的本质    
英语单词：字母长度，以及发音优选的 26进制数值；    
Colating单词：常规单词是字母长度、以及发音优选的 52+5进制数值，非常规单词是在常规单词基础之上扩展为 52+5+x进制数值。

• 学习的本质与外在表现    
掌握有效的概念总数量，    
构造不同概念流排列的逻辑关系、及其健硕程度，    
理解在不同论域内、各种概念流排列所展示出来不同的思想能量。
