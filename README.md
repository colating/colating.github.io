colating language 构词法、语法

# 科极语
- 有效的原子概念标记号的创造，永远掌握在概念创造者的手中。
- 科极语原子概念标记号的理解与传播，可以不需要第三者的介入。
- 易于理解，易于获得真知。
- 兼容在未来 AI 可能可以自行创造的概念标记号。

# 科䁒易创语（又称：科极语；音译：科拉丁语）
- 科:采用科学方法。
- 䁒：与眼、口、耳有关。
- 易：对于概念学习者而言，容易理解，容易创造出有效的概念标记号。
- 创：以人类智慧的相对无限创造能力作为服务目标。
---
