## 简介
This is a general human language.

Colating language （可乐听语）是一种新的拉丁语族通用语言文字，以英语为主要设计参考蓝本，俄语、中文、日语、韩语等为微调，并加入了 Colating自创的许多语文理论元素，从而，克服了在这些语言中所存在的不足之处。

总而言之，Colating language 语言的诞生，是建立在自成体系的通用语言文字哲学思想基础之上的产物。阅读它，可以了解一种新的通用语文，轻松理解人类通用语言文字创造的哲学原理，看懂概念级的错误释义原理，并且，还可以轻松创造属于自己的概念标识符。

• 概念的意义      
概念是人类（可能是全部地球语音动物，甚至更多）思维的基础构成单位，人类思想是建立在概念基础之上的产物。
人类群体能够掌握有效的概念总数量，是衡量不同人类时代进步程度的主要指标之一。
对于个人而言，只有掌握了足够多的有效概念，以及概念进行链接的逻辑关系，才有可能做到有效地进行思想。

• 语文系统设计的首要任务      
确定宙沏概念对象的总数量：无穷多个；
确定概念的历史寿命：非本能感知概念至少需要保留如2000年以上，本能感知概念需要保留的年限更长、甚至可以追溯到人类诞生时的原始时代；
天生本能的人类语文器官的基本能力。
注1，宙沏：宇宙内、表、外的一切。

• 概念标识符的本质      
概念图像标识符：概念对象的图像标识编码；
概念声音标识符：概念对象的声音标识编码。

• 拉丁语单词的本质      
英语单词：字母长度，以及发音优选的 26进制数值；
Colating单词：常规单词是字母长度、以及发音优选的 52+5进制数值，非常规单词是字母长度、以及发音优选的 52+5+x进制数值。

• 学习的本质与外在表现      
掌握有效的概念总数量，
概念流排列的逻辑健硕性，
在不同论域内、各种概念流排列所展示出不同的思想能量。
