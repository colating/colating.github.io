How to use Colating IME

1. Must install Rime IME input Engine.
The website of  Rime Input Method Engine, https://github.com/rime/
2. The Colating IME user manual, Use the txt editor to open the file
"colating-ime-char.schema.yaml" and see it.

3. Must install Colating font, and Colating Rime yaml file before use the IME.
The Colating font that is in the zip file, The file name such as "colating-cofang-ime-0.2.5.15".
The font file name is  "ColatingCofangSansWeb.ttf", or "ColatingCofangSans.ttf"

4. Or you can downlaod the  "c-ime-in-os" version to try.
