  简介：

     Colating Language（可乐听语 /Colating）是一种系统化创造的拉丁语族通用语言文字系统，是建立在"COLAN 人类通用语言文字创造哲学思想"之上的产物。
     1.  构词字母：由“52个标准构词字母 + 5个标准功能字母 + 5个用户自定义功能字母 + 2个淘汰存档索引功能字母”所组成。

     2.  现代理论支持：以集合论思想、模糊逻辑推理思维，以及其他现代科学理论成果， 始终贯穿于整个通用语言文字的创造过程。
     3.  概念的有效性：Colating 力求确保概念的合理集合区分，以及可接受的集合区分精度， 这是通用语文具备健硕逻辑的基础保证。 假如无法消除隐藏在构词级别的概念逻辑错误，那么，在语法、修辞， 乃至整个语文系统级别的逻辑错误将很难、甚至是无法消除。

     4.1.  消除“字词盲”：表音构词法，数量有限的优选构词字母，发音规律化，使得“见词能读，听音能写” ，从而在语文设计系统上就消除了“字词盲”。
     4.2.  消除“概念盲”：容易创造数量以亿计量级的有效“单字”，从语文系统的基础上，保证了消除概念盲成为可能。
     4.3.  “概念盲”的主要表现形式： “单字（原子概念标识符）”概念释义错误。 “单词（分子概念标识符）”概念释义与构词单元的逻辑运算结果超出了有效概念的可接受范围， 从而导致在实际应用过程中，产生不合理的概念实际理解歧义。
     4.4.  字符盲与概念盲的消除，这是一对矛与盾，应该如何来取舍与折中呢？

     5.  Colating 通用语言文字的创造原理与依据是什么呢？

     人类通用语言文字的本质：到底是什么？ 其发展动力与迭代方向的作用机理，又是怎样的呢？
     人类到底需要多少个数量的“原子概念标识符（单字）”， 才能够匹配人类的智慧能力水平呢？

     6.  阅读本网站将可获得人类通用语言文字创造在实践与探索中所产生的一些逻辑和观点， 以及Colating Language 的具体实现过程。

特点：

  1. 以哲学理论作为语文创造的基础，力求合理地设计出日常高频单体概念标识符。
  2. 容易学习、容易交流；见词能读，听音能写。
  3. 容易创造新单词、可迭代，可控的新陈代谢规则，具备健硕的逻辑运算与推理能力。
  4. 概念标识符可合理回收再利用，历史单词的归档保存。
  5. 与现代英语的兼容性：Colating 部分常用单字（类似英文自由词素）、或单词（类似英文合成词），保留了英语的发音和释义， 这样使得“Colating language基础释义典”基本上可以直接使用英文词典；但字形大多有所不同， 这是两种语言构词字母的数量与用法不一样的必定结果。
  6. 注重在实践中的实用性。

注、一些主要概念解释：

  • 概念图像标识符的分类
  1. “原子概念图像标识符”，简称“单字”：类似拉丁语的“自由词素”，或方块语文的“字”。
  2. “分子概念图像标识符”，简称“单词”：类似拉丁语的“合成词”，或方块语文的“词组”。
  3. “单体概念图像标识符”：简称“单符”：包含“原子概念标识符”、“分子概念标识符”， 它们相当于拉丁语中的“单词”，或方块语文中的“字”和“词组”。
  • “概念图像标识符”与“概念声音标识符”的关系
  1. 在同一个语文系统中，每一个概念图像标识符都是唯一的。
  2. 在Colating语文系统中，理论上，每一个概念声音标识符也是唯一的； 但是，由于有些发音很接近，因而被视为发音相同。 同音现象难以避免。相同发音的单体词，通过词性、语法规则等，是的它们可以能够被区分。
  • 概念图像标识符的合并与压缩表达式
  1. 有效概念，每一个“概念”对应于符合合理的集合划分规则的“概念对象”。
  2. 概念与概念图像标识符，合理地形成“多对一”关系，可以压缩概念图像标识符的数量， 使得概念图像标识符的学习变得简单。
  3. 概念1 → 释义条目1
    概念2 → 释义条目2 → 概念图像标识符A
    概念3 → 释义条目3
  • Colating 概念构造集合表达式
  1. 概念 → 概念图像标识符 = [{发音语法模块}]；[{概念释义模块}]；[{元集构成模块}]。

    2. 注：中括号[]，表示：可选。

  • Colating 概念释义条目表达式
  1. 概念释义条目 = [{释义最大论域}：[{跨本体概念关系}]，{对外主要表象}，{本体独有特征}]；[{设定局部论域}：{元集构成}]。

    2. 注：每一个概念对应一个概念释义条目，多个概念释义条目可能会对应于一个概念标识符相对应。

  • 概念集合表达式的化简方法
  1. 在指定了概念标识符之后：

    2. 概念 → 释义条目 → 概念图像标识符。

  2. 采用自然语言描述概念：

    3. 在特定语法论域场景的约束下：
    概念 = 概念标识符
    概念标识符@ 设定概念条目论域 = {指定的概念释义条目模块}。

  3. 采用“集合列举法”对概念进行元集描述，是概念进行逻辑运算的前提基础：

    4. 概念标识符@ 指定的概念释义条目模块 = {元集构成模块} = 集合列举法-> {元集构成}。

  • 概念的逻辑运算原理简述
  1. 除了抽象化的数字、语法概念、智灭类概念之外，宇宙中其他的一切， 理论上都可以无限细分，也就是存在有相对无限多个论域场景。
  2. 预处理的方法与步骤：对概念进行逻辑运算之前，需要进行概念预处理。
    • 按照需要讨论的具体问题，运用模糊数学原理，按照隶属度值的大小，选取和设定多个合理的论域。
    • 在设定论域中，对需要讨论的所有概念进行合理的{元集构成}分解； 并进行0级概念参考层的回归（参见Colating 概念分级与分析图表法）。
    • 按照不同的论域，利用Colating 概念分级与分析图表法，对所获得的多个逻辑运算结果进行分析， 并对所得出的多个结果进行选择，从而得出结论。
  3. 采用“集合列举法”对概念进行元集描述，是概念进行逻辑运算的前提基础：

    4. 概念标识符@ 设定概念条目论域 = {元集构成模块} = 集合列举法-> {元集构成}。

  • “集元概念盲”的两种主要表现形式：
  1. 以全盖偏

    2. 当不含空集的元集数量大于等于2时，将“概念标识符”当成其中的某个或某些元集或子集来 使用，而没有涵盖全部的非空元集加子集，从而产生了“以全盖偏”逻辑错误； 又称“白马非马”。举例：Mbs.

  2. 以偏概全

    3. 当不含空集的元集数量大于等于2时， 将某个或某些元集的“元集概念标识符” 当成 “（母集）概念标识符”来使用。 举例：Mbs.

     Colating Language 是一种经过重大改良、并具备可持续迭代改良能力、 先进的通用拉丁语族新成员。
     这是一种面向未来的人类通用型拉丁语族的语言文字。 与英语、俄语、法语、德语之类的语言一样，都是属于同一类型的通用拉丁语言文字的智慧产物。
     Colating 通过增加普通构词字母的数量，以解决拉丁语复杂的单个字母发多个不同弱规律性的音符问题 ；通过增加功能字母，以解决拉丁语中复杂的构词法问题。
     Colating 强化了通用语言的逻辑性， 同时，也顾及了自然语言表达的弹性，以及新增加了一些其他的特性；比如，历史词汇归档、可迭代性、 概念的分级与释义理解流程、概念的逻辑运算方法等等；Colating 的一些主要的语文方法和解决思路，很可能属于首创。
