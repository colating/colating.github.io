一、"Colating"是什么？

  • Colating = cosmos + latin + ing，这是一种经过重大改良、并具备可持续迭代改良能力的、先进的通用拉丁语系新成员。这是一种面向未来的人类通用型拉丁语族的语言文字。
  • 与英语、俄语、法语、德语之类的语言一样，都是属于同一类型的智慧产物。
  • Colating是一种“表音”的拉丁语族系统化创造语种。


二、Colating language（可乐听语）的特点是什么？

  • 易学习：具备健硕的构词规律和语法规律，不需要在母语环境里，采用耳濡目染的方式来学习。
  • 易交流：见词（相当于某些语言的字）能读，听音能写；
  • 易创造：从根本上杜绝拾古人的牙慧，只要想，任何人随时随地都可以创造可用于无语音障碍交流的新的colating表音单词。
  • 健硕的语言逻辑：以最大可能地避免或抑制”公说公有理，婆说婆有理“，”白马非马“的逻辑陷阱。
  • 可迭代：语言文字的学习可迭代，不会出现今天的学习对明天的学习帮助不大，甚至很小的困境；Colating语言本身的发展可迭代，具备适应人类认知概念暴增的未来环境。
  • 可新陈代谢：一种语言使用了若干千年之后，已经被淘汰不用了若干百年的高效单词可被重新释义使用。
  • 可将历史单词归档长久保存：历史像一面镜子，总会存在着可以用来比较和借鉴的地方。
  • Colating常用单词：保留了部分英语发音，但字形大多有所不同。两种语言构词字母的数量与用法不一样的必定结果。

三、对拉丁语做了哪些改进？

  • 字母增加：标准版语言用了52+3个字母，每个字母的发音具备公式化的可循规律；并且单词更加简短高效。
  • 单词的发音：包含了英语的发音，另外还有colating的专有发音字母，并且，colating的发音具有健硕的规律性可循
  • 构词法：对拉丁语中不合理的部分做了改良，增加了主要针对非日常用语、自由单词创造者、外来词汇、历史词汇等的colating功能字母构词法。
  • 功能字母：前缀、词根、后缀具备明显标识，使得非日常使用单词，具备可循规律。
  • 新增场景句法：尽可能地防止断章取义的发生。
  • 其他。

四、与现用拉丁语的衔接要点：

  • 介词、副词：发音尽量保持不变。
  • 名词：日常单词的发音，尽量保持不变。名词是Colating做出做多改变的部分，使得Colating单词具备健硕的规律性。
  • 拉丁语逻辑性强的句法：Colating语保持不变，并且Colating还增加了逻辑性强的句法。
  • 语言的模糊性：与人类的模糊逻辑思维相对应，Colating同样具有其他语言共有的模糊语法；这个问题是双刃剑，只能依靠哲学智慧与勇力来解决。

五、目前进展到哪一步了？

  • 字库：初稿已完成，并持续改进中。
  • 字母的电脑输入方法：初版已完成，并持续改进中。
  • 构词法：指导思想已初具雏形，正在进行《Colating词典》编纂与构词工作
  • 句法和语法：指导思想已初具雏形，等构词法做到某一程度之后，就可并行来完成。
  • 修辞等：还没有开始。

七、输入法：采用Rime架构进行定制。

1. COlating一共有26*2 + 3*2 + 1 + x个字母，通常使用26*2 + 3个字母。


2. 标准构词小写字母输入方法：

    前26个小写字母直接键盘输入；
    后26个小写字母" . "+键盘字母，或" ` "+键盘字母


3. 标准构词大写字母输入方法：

    前26个大写字母" , "+键盘字母;
    后26个大写字母" ; "+键盘字母;


4.  三个标准功能字母的输入方法：

    小写输入：" ' "+键盘字母m/l/p;
    大写输入：" - "+键盘字母m/l/p;


5. 三个用户自创功能字母的输入方法：

    小写输入：" ' "+键盘字母n/k/o;
    大写输入：" - "+键盘字母n/k/o;


6.  一个外来词汇标识功能字母的输入方法：

    小写输入：" ' "+键盘字母b;
    大写输入：" - "+键盘字母j;


7. 已淘汰词汇首位标识功能字母的输入方法：

    小写输入：" ' "+键盘字母r/t/y/...;
    大写输入：" - "+键盘字母r/t/y/...;

八、现阶段的强烈建议：

  • 字库和输入法，属于底层软件，coting无法保证这些文件在网络中保存和传载过程中的文件安全；因此，强烈建议有兴趣使用colating语者，直接下载虚拟硬盘.vdi文件来试用。只要导入虚拟机，就可直接试用，输入法和字库等，已经配置好了。
  • 在虚拟机中，可通过设置共享文件夹的方式来使用宿主机（安装了虚拟机VirtualBox的电脑）上的指定文件夹，这是colating gp目前所能想到的安全的、跨平台的解决方案。
  • 请 留意：colating所采用软件与系统，全都都是开源免费的，OS：Lubuntu; Office: Libre Office; 输入法：Rime；字库：adobe开源字库。当今开源软件在文档处理软件使用上的方便性，已经和Win差不多了，试用者不必过于担心使用习惯问题。

十、问与答。

1、问：为什么是在拉丁语的基础上进行改良呢？

     答：人类通用文字的数量是庞大的，并且随着时代的发展进步，新单词的数量不断在暴增着。。。。。。。
     拉丁语系能够满足见词能读、听音能写的通用语言基本要求。

2、问：为什么要创造新单词？使用已有的单词组合来表达新概念，这样不好吗？

     答：为了准确表示新发现事物和现象的需要，以及避免概念污染问题。

3、问：想要掌握常用的colating文字，需要到专门的教育机构里学习吗？

      答：在母语环境里，colating常用文字不需要到学校里学习。
            当一种语言文字设计成足够简单合理的情况下，在母语环境里，对于常用的文字，根本就不需要到专门的教育机构里面去学习就可以掌握的；其实，当今世界上早就 就存在有这样的文字，对于这一点，colating并非首创。

4、问：既然世界上早就有了在母语环境里，常用文字不需要到学校里学习的现存文字，那么。colating文字创造的意义何在？

      答：Coalting文字创造的意义：
           构词法变得更具规律性；发音变得更加简单；
           词汇的构成更具效率；增加了场景句法，在逻辑句法方面有所加强；
           正视新创词汇的重要性与可能短暂性；重视历史词汇的归档。

           Colating的这些创造与改进，使得coalting语不需要在母语环境下也能轻松掌握。人们也不需要把大量的金钱和精力花费在日常文字的学习上面。

5、问：文字的本质，是什么呢？

      答：文字的本质，其实是人类的“思想标签”；这有些类似于超市商品上面的二维码标签。
            只不过二维码标签标识的是商品的厂家、价格、出厂日期之类的信息；而文字标识的是人类各种各样的思想，以及宇宙事物或现象在人类认知中的映射。

6、问：colating为什么采用的是“表音造字法”呢？

      答：采用“表音造字法”，这完全是由人类智慧与宇宙事物或现象的无穷性决定的。
           目前只有采用“表音”的方法，才能够适应人类无穷无尽创造与发明所需要的、能够低成本进行信息交互的思想标签。
           假如人类的智慧像小鸟那样简单，那么，创造文字就是一件多此一举的事情了；
           假如人类的智慧只能容纳几十个概念，那么，可能每一个语言器官健全的人类，都是文字发明家；

           然而，
           在宇宙边界之外的无穷尽处，再往外踏出一步，哪里会有什么呢？人类能够意识到这一点，这说明人类的智慧也是无穷尽的，目前只有采用表音的方法所创造出来的新单词，才能够高效、易传播、易创造地满足“无穷尽”的宇宙与人类智慧的这一现实。

6、问：学习通用语言文字的主要方法，有哪些呢？

      答：（1）缺乏明显规律性的通用语言文字，一般都是采用“耳濡目染”的学习方式，也就是需要在母语环境里才能完成。
            （2）具有明显且有效规律性的通用语言文字，在非母语环境下，一般可以采用培训机构的方式来学习；不需要采用“耳濡目染”的母语学习方式，
